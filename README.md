# 🍔 FOOD-DEL

A full-stack food delivery web application built using the MERN stack. Users can browse food items, add items to their cart, place orders, and make online payments. The admin panel allows management of food items and orders.

## 🚀 Features

### User Features
- User Registration & Login
- Browse Food Menu
- Add/Remove Items from Cart
- Place Orders
- Online Payment Integration (Stripe)
- Order Tracking

### Admin Features
- Add New Food Items
- Update Food Details
- Delete Food Items
- View and Manage Orders
- Manage Food Listings

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- React Router
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Stripe Payment Gateway

### Admin Panel
- React.js
- Vite
- Axios
